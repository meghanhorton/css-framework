Tables

Tables include default styles for tables and data sets. Add the table class to any table element. The class will add padding, border and emphasized table header.

Striped table: Use the table-striped class to a table to add zebra striped style.
Note: <tbody> required.

Hoverable table: For hoverable table rows, you can add the table-hover class to enable hover style.

Active row: Use the active class to make <tr> element highlighted.

Forms

Sizing: To scale an input, textarea, or select box you may apply the input-sm or input-lg classes.

<!-- form select control -->
<div class="form-group">
  <select class="form-select">
    <option>Choose an option</option>
    ...
  </select>
</div>

Sizing: To scale a checkbox or radio button, you may apply the input-sm or input-lg classes.

<!-- radio buttons -->
<label class="form-radio">
  <input type="radio" name="name" checked /> 
  <span class="form-display"></span>
  Label
</label>
<!-- checkbox -->
<label class="form-checkbox">
  <input type="checkbox" name="remember"/>
  <span class="form-display"></span>
  Label
</label>
<!-- checkbox -->
<label class="form-switch">
  <input type="checkbox" name="switch"/>
  <span class="form-display"></span>
  Label
</label>
